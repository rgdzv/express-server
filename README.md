## express-server

>***<p align="justify">Express-server is a server with registration and authentication. JWT is used for authorization. You can sign up, sign in and sign out. Server is constructed with Typescript, NodeJs and SQLite to manage database. You can add additional routers, controllers and models which depends on your concern</p>***

#### What you can learn from this app:
- how to use NodeJS and Express to set up a server
- how to implement Typescript inside NodeJS
- how to create DB manually (SQLite) and initiate models
- how to set up eslint and prettier configs
- how to organize pre-commit hook to check your code before commit

#### Main used libraries:
- [typescript](https://www.typescriptlang.org)
- [NodeJS](https://nodejs.org/en)
- [sequelize](https://sequelize.org)
- [eslint](https://eslint.org)
- [prettier](https://prettier.io)

## Installation

1. Download the repo with:
```
$ git clone https://github.com/rgdzv/express-server
```
2. Install dependencies
```
$ npm install
```
3. Create a file in the root folder `database.sqlite3`

4. Run server:
```
$ npm run start:dev
```
